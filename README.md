# CLIP-Powered Modality Centering with Spiral Training for Visible-Infrared Person Re-Identification
This is the official repo of our PR paper 'CLIP-Powered Modality Centering with Spiral Training for Visible-Infrared Person Re-Identification'.

**Dataset:**
To access the new image-based visible-infrared person re-id dataset CMG-P, please send a signed [agreement](Agreement.pdf) to silverpride@foxmail.com, and we will send you the download link.
